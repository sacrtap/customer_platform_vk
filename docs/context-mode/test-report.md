## 发现的问题

### 1. Ruby iso8601 方法需要显式导入

**问题**: Ruby 执行时 `Time.now.iso8601` 报错 `undefined method`
**原因**: Ruby 标准库中 `iso8601` 需要 `require 'time'`
**解决**: 添加 `require 'time'` 后正常
**影响**: 低（用户代码中需注意导入）
