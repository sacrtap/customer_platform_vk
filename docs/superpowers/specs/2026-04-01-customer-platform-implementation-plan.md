## 1. 项目概述

### 1.1 项目目标

构建内部运营中台客户信息管理与运营系统，实现：
- 账号治理（RBAC + 自定义角色）
- 客户信息管理（基础信息 + 画像）
- 结算管理（3 种计费模式 + 余额管理 + 完整流程）
- 画像管理（双等级 + 自定义标签）
- 客户分析（四大维度 + 预测回款）

### 1.2 技术栈

| 层级        | 技术                |
| ----------- | ------------------- |
| **前端**        | Vue 3.4+ + Arco Design 2.x + Pinia + Vue Router 4 + Axios + ECharts 5 |
| **后端**        | Python 3.11 + Sanic + SQLAlchemy 2.0 + Alembic + APScheduler |
| **数据库**      | PostgreSQL 15 |
| **缓存**        | Redis 7 (可选) |
| **容器化**      | Podman + podman-compose |
| **文件存储**    | 本地存储 / MinIO |

### 1.3 开发周期

**总周期**: 13 周（约 3 个月）

**阶段划分**:
| 阶段    | 周期       | 主题               |
| ------- | ---------- | ------------------ |
| Phase 0 | Week 1     | 项目初始化         |
| Phase 1 | Week 2-3   | 用户与权限模块     |
| Phase 2 | Week 4-5   | 客户信息管理模块   |
| Phase 3 | Week 6-8   | 结算管理模块       |
| Phase 4 | Week 9     | 标签管理模块       |
| Phase 5 | Week 10-11 | 客户分析模块       |
| Phase 6 | Week 12    | 定时任务与外部接口 |
| Phase 7 | Week 13    | 集成测试与部署     |
