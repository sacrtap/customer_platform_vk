## 工作流说明

1. **Spec 先行**: 每个功能先创建设计规格 (`specs/`)
2. **Plan 跟进**: 基于规格拆解为实现计划 (`plans/`)
3. **执行跟踪**: 计划中的任务按 Phase 逐步执行
4. **归档管理**: 已完成的计划保持归档，便于追溯
