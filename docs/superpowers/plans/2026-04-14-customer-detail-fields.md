> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在客户详情页面的基础信息和画像信息 Tab 中补充 12 个核心业务字段（8 个 customers 表字段 + 4 个 customer_profiles 表字段）

**Architecture:** 按照自底向上的顺序实施：数据库迁移 → Model 层 → Service 层 → API 路由 → TypeScript 类型 → 前端展示 → 前端编辑。每个字段新增均为 nullable，不影响现有数据。

**Tech Stack:** Python/SQLAlchemy/Alembic (后端), Vue 3/TypeScript/Arco Design (前端)
