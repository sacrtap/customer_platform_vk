## 系统概述

客户运营中台是用于公司内部运营管理的系统，主要功能包括：

- **客户信息管理**：维护客户基础信息、画像数据
- **标签管理**：自定义标签体系，支持客户打标和组合筛选
- **结算管理**：计费规则配置、结算单生成、余额管理
- **客户分析**：消耗分析、回款分析、健康度分析、画像分析

### 系统角色

| 角色 | 权限说明 |
|------|----------|
| 系统管理员 | 账号管理、角色权限配置、系统设置 |
| 运营经理 | 客户管理、结算管理、定价配置、全局数据查看 |
| 销售人员 | 客户跟进、业务管理 |
| 数据分析师 | 客户分析、报表查看 |

## 登录与退出

### 登录系统

1. 打开浏览器，访问系统地址（如：http://localhost:5173）
2. 输入用户名和密码
3. 点击"登录"按钮

> 默认管理员账号：`admin` / `admin123`（首次登录后请修改密码）

### 退出系统

1. 点击右上角头像
2. 选择"退出登录"

## 客户管理

### 查看客户列表

1. 点击左侧菜单"客户管理"
2. 可使用筛选条件：关键词、账号类型、业务类型、客户等级等
3. 点击"查询"按钮刷新列表

### 新建客户

1. 点击"新建客户"按钮
2. 填写客户信息：
   - 公司 ID（必填，唯一标识）
   - 客户名称（必填）
   - 账号类型、业务类型、客户等级
   - 结算方式、结算周期
   - 是否重点客户
   - 邮箱地址
3. 点击"确定"保存

### 编辑客户

1. 在客户列表中找到目标客户
2. 点击"编辑"按钮
3. 修改客户信息
4. 点击"确定"保存

### 客户详情

1. 点击客户列表中的"详情"按钮
2. 查看客户完整信息，包括：
   - 基础信息
   - 客户画像
   - 账户余额
   - 客户标签

### Excel 导入客户

1. 点击"导入客户"按钮
2. 下载 Excel 模板
3. 填写客户数据
4. 上传 Excel 文件
5. 查看导入结果

### Excel 导出客户

1. 在客户列表页面点击"导出"按钮
2. 系统自动生成 Excel 文件并下载

## 标签管理

### 查看标签列表

1. 点击左侧菜单"标签管理"
2. 可查看标签名称、类型、分类等信息

### 新建标签

1. 点击"新建标签"按钮
2. 填写标签信息：
   - 标签名称（必填）
   - 标签类型：客户标签/画像标签
   - 标签分类（可选）
3. 点击"确定"保存

### 给客户打标

1. 进入客户详情页
2. 切换到"客户标签"标签页
3. 点击"添加标签"
4. 选择要添加的标签
5. 点击"确定"

### 移除标签

1. 在客户详情页的标签列表中
2. 点击标签右侧的"×"按钮
3. 确认移除

### 标签组合筛选

1. 在客户列表页面点击"高级筛选"
2. 选择一个或多个标签
3. 点击"查询"查看匹配的客户

## 结算管理

### 余额管理

#### 查看余额列表

1. 点击左侧菜单"结算管理" → "余额管理"
2. 查看各客户的余额情况

#### 客户充值

1. 点击"充值"按钮
2. 选择客户
3. 填写充值金额：
   - 实充金额
   - 赠费金额（可选）
   - 备注
4. 上传支付凭证（可选）
5. 点击"确定"

### 定价规则

#### 查看定价规则

1. 点击"结算管理" → "定价规则"
2. 查看各客户的计费规则配置

#### 新建定价规则

1. 点击"新建规则"按钮
2. 选择客户
3. 配置计费信息：
   - 设备类型：X/N/L 系列
   - 计费类型：定价结算/阶梯结算/包年结算
   - 单价或阶梯配置
   - 生效日期
4. 点击"确定"保存

### 结算单管理

#### 查看结算单

1. 点击"结算管理" → "结算单"
2. 可查看结算单列表，支持按状态筛选

#### 生成结算单

1. 点击"生成结算单"按钮
2. 选择客户（可多选）
3. 选择结算周期
4. 添加结算项：
   - 设备类型
   - 图层类型
   - 数量
   - 单价
5. 点击"生成"

#### 结算单状态流转

结算单状态流转顺序：

```
草稿 → 已提交 → 已确认 → 已付款 → 已完成
```

- **草稿**：运营经理创建结算单
- **已提交**：商务确认提交
- **已确认**：客户外部系统确认
- **已付款**：商务上传付款凭证
- **已完成**：系统扣款完成

#### 应用减免

1. 在结算单详情页，状态为"草稿"时
2. 点击"减免"按钮
3. 填写减免金额和原因
4. 上传减免证明附件
5. 点击"确定"

## 常见问题

### Q: 忘记密码怎么办？

A: 请联系系统管理员重置密码。

### Q: 如何批量给客户添加标签？

A: 目前支持在客户详情页逐个添加，批量操作功能开发中。

### Q: 结算单生成后如何修改？

A: 状态为"草稿"的结算单可以直接编辑，已提交的结算单需要联系管理员。

### Q: 如何查看客户的消费记录？

A: 在客户详情页切换到"账户余额"标签页，可查看消费流水。

### Q: Excel 导入失败怎么办？

A: 请检查：
1. Excel 文件格式是否正确
2. 必填列是否完整
3. 数据格式是否符合要求
4. 公司 ID 是否重复
