# 客户运营中台 - 项目总结报告

## 项目概述

**项目名称**: 客户运营中台 (Customer Platform VK)

**开发周期**: 13 周 (2026-04-01 至 2026-04-02)

**项目目标**: 构建内部运营中台客户信息管理与运营系统，实现账号治理、客户信息管理、结算管理、画像管理、客户分析。

---

## 技术栈

| 层级 | 技术选型 |
|------|----------|
| **前端** | Vue 3.4+ + TypeScript + Arco Design 2.x + Pinia + Vue Router 4 + ECharts 5 |
| **后端** | Python 3.11 + Sanic + SQLAlchemy 2.0 + Alembic + APScheduler |
| **数据库** | PostgreSQL 15 |
| **缓存** | Redis 7 |
| **容器化** | Podman + podman-compose |
| **监控** | Prometheus + Grafana |
| **测试** | pytest + pytest-asyncio + pytest-cov + locust |

## 关键特性

### 1. 余额管理
- 实充金额 + 赠费金额分离
- 先消耗赠费，再消耗实充
- 余额不足预警

### 2. 结算单状态机
```
草稿 → 已提交 → 已确认 → 已付款 → 已完成
```

### 3. 计费模式
- **定价结算**: 按设备类型配置单层/多层定价
- **阶梯结算**: 按用量阶梯计价
- **包年结算**: A/B/C/D 套餐

### 4. 标签体系
- 客户标签 + 画像标签双体系
- 支持自定义标签
- 标签组合筛选

### 5. 定时任务
- 5 个自动化任务
- 执行日志记录
- 失败重试机制
