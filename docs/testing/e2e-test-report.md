## 运行说明

### 前置条件

1. **后端服务运行**: http://localhost:8000
2. **前端服务运行**: http://localhost:5173
3. **测试数据**: 数据库包含测试账号和客户数据

### 快速运行

```bash
# 方式 1: 使用自动化脚本
./scripts/run-e2e-tests.sh

# 方式 2: 手动运行
cd frontend
npx playwright test --project=chromium

# 方式 3: UI 模式 (推荐调试)
npx playwright test --ui
```

### 运行特定测试

```bash
# 仅登录测试
npx playwright test test_login_flow.spec.ts

# 仅客户管理测试
npx playwright test test_customer_crud.spec.ts

# 仅结算单测试
npx playwright test test_invoice_workflow.spec.ts

# 仅余额测试
npx playwright test test_balance_recharge.spec.ts
```
