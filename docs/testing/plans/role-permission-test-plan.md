**文档版本**: v1.0

## 一、测试范围

### 1.1 被测功能

角色权限管理页面 (`/roles`)，包含以下核心功能：
- 角色列表展示与分页
- 角色搜索
- 角色 CRUD（创建、读取、更新、删除）
- 权限配置（权限树、选择/取消选择、全选/取消全选）
- 系统角色保护机制

### 1.2 涉及模块

| 层级   | 技术栈                        | 文件路径                                      |
| ------ | ----------------------------- | --------------------------------------------- |
| 前端   | Vue 3.4 + TypeScript 5.3      | `frontend/src/views/roles/Index.vue`          |
| 前端   | Arco Design 2.54              | `frontend/src/api/roles.ts`                   |
| 后端   | Python 3.12 + Sanic 22.12     | `backend/app/routes/roles.py`                 |
| 后端   | SQLAlchemy 2.0 + PostgreSQL   | `backend/app/services/roles.py`               |
| 后端   | 数据模型                      | `backend/app/models/users.py`                 |
| 后端   | 权限路由                      | `backend/app/routes/permissions.py`           |

### 1.3 测试类型

- ✅ 功能测试
- ✅ UI/UX 测试
- ✅ 集成测试
- ✅ 边界测试
- ✅ 安全测试（权限校验、系统角色保护）
- ✅ API 测试

## 二、功能模块分解

### 2.1 页面展示功能

| 功能点       | 说明                                          | 优先级 |
| ------------ | --------------------------------------------- | ------ |
| 页面标题     | 显示"角色权限"和副标题"角色管理与权限配置"    | P0     |
| 搜索框       | 支持按角色名或描述搜索                        | P1     |
| 新建角色按钮 | 打开新建角色对话框                            | P0     |
| 角色列表表格 | 展示 ID、角色名称、描述、类型、创建时间、操作 | P0     |
| 分页组件     | 支持页码切换和每页数量调整                    | P1     |

### 2.2 角色 CRUD 功能

| 功能点   | 说明                               | 优先级 |
| -------- | ---------------------------------- | ------ |
| 创建角色 | 输入名称和描述，创建新角色         | P0     |
| 编辑角色 | 修改自定义角色的名称和描述         | P0     |
| 删除角色 | 删除自定义角色（系统角色不可删除） | P0     |
| 查看列表 | 分页展示角色列表                   | P0     |
| 搜索角色 | 按关键词过滤角色                   | P1     |

### 2.3 权限配置功能

| 功能点        | 说明                           | 优先级 |
| ------------- | ------------------------------ | ------ |
| 打开权限配置  | 点击"权限配置"按钮打开对话框   | P0     |
| 权限树展示    | 按模块分组展示权限（树形结构） | P1     |
| 选择权限      | 勾选/取消勾选权限              | P0     |
| 全选/取消全选 | 一键选择/取消所有权限          | P1     |
| 保存权限      | 提交权限配置到后端             | P0     |
| 已选权限计数  | 显示已选择的权限数量           | P2     |
