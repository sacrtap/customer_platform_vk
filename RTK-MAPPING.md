# RTK 命令映射表

**最后更新**: 2026-04-11  
**适用环境**: OpenCode + RTK (Rust Token Killer)

---

## 核心原则

### 混合模式规则

| 场景 | 工具选择 | 示例 | Token 节省 |
|------|----------|------|----------|
| 快速探索/多文件搜索 | 内置 `grep`/`glob`/`read` | 并行执行，结构化输出 | - |
| 大输出搜索/日志分析 | `bash` + `rtk grep`/`rtk log` | `rtk grep -r "pattern" .` | 75% |
| Git/测试/构建命令 | `bash` + `rtk` 前缀 | `rtk git status` | 59-99% |
| 精确单文件操作 | 内置工具 | 直接 `read` 文件 | - |

## RTK 命令完整映射

### 1. 文件搜索 (60-75% 节省)

| 内置工具 | RTK Bash 命令 | 使用场景 |
|----------|---------------|----------|
| `grep(pattern="x")` | `rtk grep -r "x" .` | 大输出搜索 |
| `grep(pattern="x", include="*.ts")` | `rtk grep -r "x" . --include="*.ts"` | 按类型搜索 |
| `glob(pattern="**/*.ts")` | `rtk find . -name "*.ts"` | 文件查找 |
| `read(filePath)` | `rtk read file.ts` | 代码阅读 |
| - | `rtk ls -la src/` | 目录列表 |

**示例**:
```bash
# 搜索所有认证相关代码
rtk grep -r "auth_required" . --include="*.py"

# 查找所有 TypeScript 测试文件
rtk find . -name "*.spec.ts"

# 阅读配置文件
rtk read backend/app/config.py
```

| 推送远程 | `rtk git push` | - |
| 拉取更新 | `rtk git pull` | - |
| 分支列表 | `rtk git branch` | 65% |
| 工作树 | `rtk git worktree list` | 70% |

**示例**:
```bash

rtk git add . && rtk git commit -m "feat: add new feature" && rtk git push

# 查看最近 5 次提交
rtk git log -5 --oneline

# 对比工作区与暂存区
rtk git diff HEAD
```

## 最佳实践

### ✅ 推荐

```bash
# 链式命令，每个都加 rtk
rtk git add . && rtk git commit -m "msg" && rtk git push

# 大输出搜索
rtk grep -r "TODO" . --include="*.py" | head -20

# 测试仅看失败
rtk pytest --tb=line -q
```

### ❌ 避免

```bash
# 忘记 rtk 前缀
git status  # 应：rtk git status

# 链式命令只加一个 rtk
rtk git add . && git commit -m "msg"  # 应：每个命令都加

# 在 AGENTS.md 限制的场景使用 curl/wget
curl https://api.example.com  # 应：context-mode_ctx_fetch_and_index
```
